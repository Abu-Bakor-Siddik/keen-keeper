Project name : Keen Keeper

Description:
A personal relationship management app to help you stay connected with the people who matter most — track friends, log interactions, and never let an important relationship slip away.

Technologies:
1. Next.js
2. Raect
3. Tailwind CSS
4. DaisyUI
5. React Toastify
6. React Icons
7. Recharts
8. JavaScript (ES6+)
9. JSON (for product data)

Features:
1. Friend Profiles — View each friend's contact history, relationship goal, tags, and status (On-Track / Almost Due / Overdue)
2. Quick Check-In — Log a Call, Text, or Video interaction from any friend's page with an instant toast confirmation
3. Filterable Timeline — Browse your full interaction history, filterable by type (Call / Text / Video) across all friends
